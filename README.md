ペイパル決済モジュール挙動修正プラグイン
==================================

本プラグインについて
-----------------

EC-CUBE2用のペイパル決済モジュール挙動修正プラグインです。


修正項目
-------

以下の挙動を修正します。

メール

- テンプレートディレクトリとして、以下のパスを追加します。
    - TEMPLATE_REALDIR . 'dropped_items_noticer/'


EC-CUBE2 CLI
------------

以下のコマンドを追加します。

`paypal-express:copy`
